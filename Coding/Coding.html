For the curious:

The robot in the animation panel has a few sensors on board: a distance sensor, and an array of 8 line sensors. The two programs shown use these sensors to do autonomous tasks. The wall-following program uses the distance sensor to crawl along whatever obstacles are to the robot's right side. The line-following program contains similar code to use its array of 8 line sensors (which turn tan when over a line) to move the robot forward over the blue line. Hitting the 'load' button will load onto the robot whatever program is currently visible on the tabbed panel.
